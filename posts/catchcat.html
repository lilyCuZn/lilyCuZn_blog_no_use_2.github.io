原理

网站有前端和后端相互支持（当然本站用 GitPage 没有后端）。简单说一下前端，用户可见的界面叫前端，有三个技术

  1. HTML 标记语言，即网页内容
  2. CSS 布局，定义 HTML 元素怎么显示
  3. JS 控制网页行为

以上若有错误，还请斧正。
